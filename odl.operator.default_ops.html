default_ops¶

Default operators defined on any (reasonable) space.

Classes

ComplexEmbedding(space[, scalar]) Operator that embeds a vector into a complex space.
ComplexModulus(space) Operator that computes the modulus (absolute value) of a vector.
ComplexModulusSquared(space) Operator that computes the squared complex modulus (absolute value).
ConstantOperator(constant[, domain, range]) Operator that always returns the same value.
DistOperator(vector) Operator taking the distance to a fixed space element.
IdentityOperator(space) Operator mapping each element to itself.
ImagPart(space) Operator that extracts the imaginary part of a vector.
InnerProductOperator(vector) Operator taking the inner product with a fixed space element.
LinCombOperator(space, a, b) Operator mapping two space elements to a linear combination.
MultiplyOperator(multiplicand[, domain, range]) Operator multiplying by a fixed space or field element.
NormOperator(space) Vector space norm as an operator.
PowerOperator(domain, exponent) Operator taking a fixed power of a space or field element.
RealPart(space) Operator that extracts the real part of a vector.
ScalingOperator(domain, scalar) Operator of multiplication with a scalar.
ZeroOperator(domain[, range]) Operator mapping each element to the zero element.
